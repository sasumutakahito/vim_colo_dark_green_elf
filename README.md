## INSTALL:

1) Get the source

`git clone https://github.com/sasumutakahito/vim_colo_dark_green_elf.git`

2) Put in Your ~/.bashrc

`export TERM=xterm-256color`

`export TERMCAP=`

3) Put file elf.vim in Your ~/.vim/colors

4) Put in Your ~/.screenrc:

`term screen-256color`

5) Put in Your ~/.vimrc 

`colo elf`

   or test using

`colo elf`

   during editing
